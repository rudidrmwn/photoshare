## ðŸ“¦ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/rudidrmwn/photoshare.git
   cd photoshare
   ```
2. ** Docker Compose **
   ```bash
   docker compose up -d //Start Docker
   docker compose down  //Stop  Docker
   ```

3. ** Backend **
   ```bash
   cd backend
   npm run start
   ```

4. ** Frontend **
   ```bash
   cd frontend
   npm run dev
   ```

5. ** Seed Dummies Data **
   ```bash
   cd  backend
   node seed.js
   node seedusers.js
   ```
